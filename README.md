# Advanced BMI Calculator
A professional and user-friendly BMI (Body Mass Index) Calculator application built with Python and Tkinter. This tool allows users to calculate their BMI using both Metric and Imperial units, provides gender-specific analysis, and visualizes the results.

# 🚀 Key Features
Dual Measurement Systems: Switch seamlessly between Metric (kg/m) and Imperial (ft/in) units.

Gender-Specific Analysis: Tailored insights based on user gender.

Visual Chart: Includes a built-in BMI classification chart to help users understand their health status.

Data Logging: Automatically logs calculation history locally for future reference.

User-Friendly GUI: Clean, intuitive interface with clear color-coded actions (Calculate, Reset, Exit).

# 🛠 Tech Stack
Language: Python

GUI Framework: Tkinter

Tools: Git & GitHub (Version Control)
